Deletion Tools

This page summarizes the administration actions available in the system. To delete a club or an activity, use the delete buttons that appear in the lists of the Clubs and Activities pages.

What you can do from the Admin Panel:

  • Delete any club
  • Delete any activity
  • Manage data through the API or LocalStorage (fallback mode)

This is a simplified administrator interface meant to showcase backend interactions and basic CRUD operations.
